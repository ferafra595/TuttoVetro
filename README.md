# Tutto Vetro - sito statico

Sito statico pronto per GitHub + Cloudflare Pages.

## Struttura
- `index.html` → home page
- `cataloghi.html` → pagina cataloghi
- `chi-siamo.html` → presentazione attività
- `contatti.html` → recapiti diretti
- `assets/css/style.css` → stile
- `assets/js/script.js` → menu mobile
- `assets/catalogs/catalogo-1.pdf` → PDF Catalogo 1
- `assets/catalogs/catalogo-2.pdf` → PDF Catalogo 2

## Cosa devi fare
1. Sostituisci i file PDF segnaposto dentro `assets/catalogs/` con i tuoi file reali.
2. Carica tutto su un repository GitHub.
3. Collega il repository a Cloudflare Pages.

## Deploy Cloudflare Pages
- Framework preset: `None`
- Build command: lascia vuoto
- Build output directory: `/`

## Nota
Il sito è statico, quindi non richiede backend.
